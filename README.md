# CMRBolsa One Page + páginas internas

Estructura correcta:

- La navbar principal navega por anclas dentro de la Home: `#inicio`, `#cmrbolsa`, `#comunidad`, etc.
- Dentro de cada sección hay botones que llevan a páginas internas completas: `/comunidad-pev`, `/mentoria-pev`, etc.

## Ejecutar

```bash
npm i
npm run dev
```

## Configuración

Edita URLs de Cloudinary, Comudia, Substack y formularios en:

`src/data/siteConfig.js`
